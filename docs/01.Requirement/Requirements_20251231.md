# RedArrow - 주식 단타 자동 매매 시스템 요구사항

## 문서 정보
- **최초 작성일**: 2025-12-31
- **최종 수정일**: 2025-12-31
- **버전**: 1.0

---

## 프로젝트 개요
주식 단타 자동 매매를 위한 독립적인 시스템으로, 시뮬레이션과 실제 트레이딩을 모두 지원하는 프로그램

---

## 요구사항 목록

### 1. 시스템 기본 요구사항 (2025-12-31 추가)

#### 1.1 운영 모드
- 시뮬레이션 모드 지원
- 실제 트레이딩 모드 지원

#### 1.2 프로그램 실행 방식
- 독립적인 프로그램으로 동작
- 주식 개장 시간부터 종료 시간까지 자동 실행
- 시장 상황에 따른 자동 시작/종료

---

### 2. 종목 선정 프로세스 (2025-12-31 추가)

#### 2.1 룰셋 생성
- 여러 Resource를 활용한 데이터 수집
- Google NotebookLM을 활용한 룰셋 생성
- 룰셋 정의 및 관리 기능

#### 2.2 데이터 수집
- 정의된 룰셋에 맞춘 데이터 수집
- 실시간 데이터 처리
- 과거 데이터 분석

#### 2.3 종목 선정
- 룰셋 기반 종목 필터링
- 선정 기준 충족 여부 판단
- 선정된 종목 리스트 관리

---

### 3. 자동 매매 시스템 (2025-12-31 추가)

#### 3.1 매매 룰셋
- 목표 수익률 설정
- 손실 제한(Stop Loss) 설정
- 룰셋 기반 자동 매매 실행

#### 3.2 매매 실행
- 자동 주문 체결
- 실시간 포지션 모니터링
- 목표 달성 시 자동 청산
